## Conversor de Moedas – Real Time

Um conversor de moedas moderno com suporte a bandeiras, interface elegante e atualização em tempo real via API pública.

## Preview

- Conversão instantânea entre várias moedas

- Bandeiras nos seletores

- Taxas atualizadas automaticamente pela API

- Interface simples, moderna e responsiva



## Funcionalidades

- Conversão entre diversas moedas

- Taxas obtidas em tempo real via ExchangeRate API

- Bandeiras nos seletores de moeda

- Botão de inverter moedas

- Atualização automática do valor final

- Interface moderna e limpa


## Tecnologias Utilizadas

- HTML5

- CSS3

- JavaScript (Vanilla JS)

- ExchangeRate API (dados reais)

## ▶ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/analacerda-dev/conversor-moedas.git
```

2. Entre na pasta:

```
cd conversor-moedas
```

3. Abra o arquivo no navegador:

```
open index.html
```

ou simplesmente dê dois cliques no arquivo `index.html`.
